## 레이아웃 개요
- **페이지 헤더**: "내 에이전트" 제목과 새 에이전트 생성 버튼이 상단에 위치합니다.
- **요약 바**: 전체/공개/비공개/초안 개수를 한눈에 보여주는 통계 막대가 제공됩니다.
- **검색/필터**: 키워드 검색과 카테고리(튜터·콘텐츠·번역·데이터 분석 등) 필터를 조합해 원하는 에이전트를 찾습니다.
- **에이전트 카드**: 제목, 설명, 태그, 평점/사용량/즐겨찾기 수, 공개 범위, 소유자 정보를 포함하며 카드 내 액션 버튼으로 상세 보기, 복제, 공유, 삭제가 가능합니다.

## 주요 사용 흐름

### 새 에이전트 생성
1. 상단 **"새 에이전트"** 버튼 클릭.
2. 템플릿 선택 또는 빈 상태로 시작 후 시스템 프롬프트/도구/예시 대화를 구성합니다.
3. 저장 시 공개 범위(전체/클래스/개인)를 지정하고, 필요하면 즐겨찾기에 고정합니다.

### 목록 탐색과 관리
- **검색창**으로 이름·키워드·설명에 포함된 텍스트를 즉시 필터링합니다.
- **카테고리 태그**(예: "프로그래밍 튜터", "콘텐츠 작성", "번역", "데이터 분석")를 눌러 분야별로 좁혀봅니다.
- 카드 우측 상단 액션으로 **상세 보기/복제/공유/삭제**를 실행하고, 카드 내 토글로 **공개/비공개** 상태를 전환합니다.
- 평가 지표(별점·사용량·북마크 수)를 통해 인기 에이전트를 빠르게 파악합니다.

### 공유와 협업
- **공개 설정**을 "전체"로 두면 다른 학습자도 카드 목록에서 바로 사용할 수 있습니다.
- **클래스 한정 공유** 시 특정 수업의 실습 과제로 연결할 수 있으며, 운영자는 클래스별 공유 현황을 대시보드에서 확인합니다.
- **복제(포크)** 후 수정하면 원본을 유지한 채 맞춤형 버전을 만들 수 있어, 강사용 템플릿 → 학습자 개인화 흐름을 지원합니다.
