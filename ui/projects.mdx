## 페이지 구성
- **헤더**: "교육 프로젝트" 제목과 "새 프로젝트" 버튼이 배치되어 즉시 생성이 가능합니다.
- **필터 바**: 클래스 선택 드롭다운, 상태(진행 중/종료), 정렬(최신순/이름순/수강생 수) 및 검색창을 제공합니다.
- **프로젝트 카드**: 제목·설명·담당자·상태 뱃지 외에 수강생 수, 과제 수, 최근 업데이트, 참여율 등의 지표가 포함됩니다.
- **액션 버튼**: 카드 내에서 바로 상세 보기, 복제, 공유 링크 복사, 삭제를 실행할 수 있습니다.

## 워크플로우

### 새 프로젝트 생성
1. 상단 **"새 프로젝트"** 버튼 클릭 후 프로젝트 이름/설명을 입력합니다.
2. 담당자와 클래스 범위를 지정하고, 필요 시 기본 과제/자료를 추가합니다.
3. 저장하면 목록에 카드가 생성되고 기본 필터(최신순)에 따라 정렬됩니다.

### 프로젝트 탐색
- **클래스 필터**로 특정 반/기수를 선택하고, **상태 필터**로 진행 중/종료 프로젝트를 분리해 봅니다.
- 카드의 **지표 요약(수강생/과제/참여율)**을 확인해 학습 참여 상황을 파악합니다.
- **검색창**으로 제목/설명에 포함된 키워드 기반 검색을 수행합니다.

### 카드 액션 활용
- **상세 보기**: 프로젝트 대시보드로 이동해 과제/실습 현황을 점검합니다.
- **복제**: 동일한 템플릿을 다른 클래스에 재활용할 때 사용합니다.
- **공유**: 강사용 초대 또는 뷰 전용 링크를 복사해 협업을 시작합니다.
- **삭제**: 불필요한 프로젝트는 즉시 삭제하고, 필요 시 토스트 알림으로 결과를 확인합니다.
